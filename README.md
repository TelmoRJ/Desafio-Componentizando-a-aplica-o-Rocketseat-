## Como testar?
1 - Clone o Projeto
```
git clone <url>
```
2 - Acesse a Pasta do Repositorio
```
cd <Caminho da Pasta>
```

3 - Instale as Dependencies
```
npm i
```

4 - Rode o Comando
```
npm run dev
```

5 - Abre um novo Terminal e Rode o Comando
```
npm run server
```
